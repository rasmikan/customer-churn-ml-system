# Customer Churn ML System

## Overview
An end-to-end machine learning system to predict telecom customer churn.

## Features
- Real-time churn prediction
- Batch prediction using CSV
- Customer risk segmentation
- Interactive analytics dashboard

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- Plotly

## Model
- Logistic Regression
- Random Forest
- XGBoost

## Run Locally
pip install -r requirements.txt
cd app
streamlit run app.py
